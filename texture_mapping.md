# 程序贴图生成

## 时间
2026-04-18 14:33

## 内容
为梦想小屋添加程序生成的纹理贴图，无需外部图片文件。

## 贴图类型

### 1. 砖墙 (brickTex)
- 256x256 Canvas
- 交错排列的砖块（45x18px）
- 随机明暗变化模拟真实砖块
- 灰浆缝隙

### 2. 木纹 (woodTex)
- 竖向木纹线条
- 年轮/节眼效果
- 轻微噪声增加质感

### 3. 瓦片屋顶 (roofTex)
- 鱼鳞状瓦片排列
- 随机明暗
- 深色勾边

### 4. 草地 (grassTex)
- 3000+ 草叶像素点
- 深浅斑块渐变

### 5. 石板 (stoneTex)
- 矩形石板（50x50）
- 交错排列
- 灰缝勾边

## 应用位置
- 地面：草地
- 院子格子：草地
- 石板路：石板
- 墙体：砖墙
- 屋顶：瓦片
- 门：木纹
- 篱笆：木纹

## 技术实现
- Canvas 2D API 生成
- THREE.CanvasTexture 转换
- RepeatWrapping 重复平铺
- repeat.set() 调整平铺次数
