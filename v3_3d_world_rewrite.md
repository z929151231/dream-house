# 梦想小屋 v3 — 完整3D世界重写

## 时间
2026-04-18 11:26

## 目标
将游戏从"分区域的2D格子"改为"完整3D世界"——房子是真正的3D建筑，院子围绕房子展开，可360°自由旋转。

## 核心改动

### 场景架构
- **旧版**：正交相机 + 6个独立区域切换 + 小格子地面
- **新版**：透视相机 + 一个完整大世界 + 房屋居中 + 院子环绕

### 相机系统
- `PerspectiveCamera`（透视，真实3D感）
- 球坐标轨道控制：左键拖拽旋转，右键平移，滚轮缩放
- 触屏双指缩放支持
- 俯仰角限制：0.15~1.45（防止穿地/翻顶）

### 房屋
- 真正的3D建筑：四面墙 + 八个窗户 + 门 + 门廊柱 + 台阶 + 四坡屋顶 + 烟囱
- 5个等级：小木屋→砖瓦房→欧式别墅→现代豪宅→梦幻城堡
- 夜晚窗户发光效果

### 院子
- 房屋中心3x3=9格不可放置（房子占地）
- 外圈9x9-9=72格可放置装饰物
- 圆形篱笆围墙包围
- 前门石板小路延伸

### 保留功能
- 23种商店物品（树/花/水/装饰/地面）
- 昼夜循环 + 天空颜色变化
- 萤火虫（夜晚）+ 樱花粒子
- 物品生长动画 + 成熟闪烁
- 风车旋转、猫咪摇尾、天使光环

## 修复的bug
- clock.getDelta() vs getElapsedTime() 冲突
- 屋顶顶点颜色→computeVertexNormals + DoubleSide
- 萤火虫每帧重建→状态标志控制
- mouse/raycaster 初始化
- 可选链兼容性
